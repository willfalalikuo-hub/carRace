# 汽车跑酷游戏

一款基于 HTML5 Canvas 的街机风格赛车游戏。选择你的车辆，在多车道公路上躲避其他车辆，挑战最高分！

## 功能特性

- **4种可选车辆**: 跑车、轿车、卡车、SUV，各有不同的速度和操控特性
- **动态障碍物**: 随机生成的其他车辆，以较慢速度行驶，会随机变道
- **持续前进效果**: 即使不加速，道路也会持续向前移动
- **碰撞反馈**: 屏幕震动和红色闪烁效果，增强碰撞体验
- **街机风格音效**: 使用 Web Audio API 合成的背景音乐和碰撞音效
- **计分系统**: 基于速度和距离的动态计分
- **流畅动画**: 60fps 游戏循环，流畅的道路和车辆动画

## 游戏机制

### 碰撞反馈系统
- 碰撞时播放碰撞音效
- 屏幕震动效果，增强冲击感
- 红色闪烁效果，视觉反馈明显
- 震动和闪烁逐渐衰减，自然流畅

### 碰撞抵抗系统
- 不同车辆拥有不同的碰撞抵抗能力
- 跑车和轿车：无碰撞抵抗，碰撞即游戏结束
- SUV：可抵抗1次碰撞，碰撞后护盾消失
- 卡车：可抵抗2次碰撞，碰撞后护盾减少
- 碰撞抵抗会在游戏开始时根据选择的车辆自动设置

### 障碍物生成
- 障碍物为其他车辆，在 4 条车道中随机生成
- 具有车道碰撞检测，避免生成无法躲避的障碍
- 障碍物以较慢的速度（0.5-1）向下移动
- 障碍物会随机变道，增加游戏难度
- 随着分数增加，障碍物生成频率加快

### 计分规则
```
分数 = 基础分 + (速度 * 0.2)
```

速度越快，得分越高。

### 游戏视角
- 玩家车辆在屏幕底部固定位置
- 道路和障碍物持续向下移动，模拟前进效果
- 即使不加速，道路也会以基础速度向前移动
- 加速会增加道路移动速度
